# Overview

OpenGL playground using Go.

## Preparations

    brew install glfw3
    cd /usr/local/lib
    ln -s ../Cellar/glfw/3.3.2/lib/libglfw.3.3.dylib libglfw3.dylib
    
    go get github.com/markbates/pkger/cmd/pkger
